# Installation-Linux-Termux

```
apt-get update && apt-get upgrade && apt-get install git python3 nodejs
```
```
git clone https://github.com/AnukarOP/Multi-CC
```
```
cd Multi-CC
```
```
pip3 install -r requirements.txt
```
```
python3 cc.py
```
